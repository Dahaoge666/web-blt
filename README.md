# 博利通精密工艺官网（轻量版）

基于 **Vite + 原生 HTML/CSS** 的企业展示站初始化项目，面向机械加工与表面处理行业场景。

## 站点定位

- 企业品牌：**博利通（BOLITONG）**
- 行业方向：机械加工、工业表面处理
- 主营工艺：**电泳加工、喷漆加工、喷涂与后处理**
- 页面目标：展示企业实力、主营业务、标准流程与合作入口

## 快速启动

```bash
npm install
npm run dev
```

默认地址：`http://localhost:5173`

## 常用命令

- `npm run dev`：启动开发环境
- `npm run build`：构建生产版本
- `npm run preview`：预览构建结果
