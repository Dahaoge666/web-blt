# AGENTS.md

本文件是本仓库的 AI 协作上下文入口，供后续智能体/开发者快速了解项目并遵循统一规范。

## 1. 项目背景

- 项目名称：博利通精密工艺官网（轻量版）
- 业务方向：机械加工与工业表面处理
- 主要工艺：电泳、喷漆、喷粉、后处理
- 目标：以低维护成本快速呈现企业实力、业务能力、工艺流程与合作入口

## 2. 技术栈

- 构建/开发：Vite
- 前端：原生 HTML + CSS（无框架）
- 运行环境：Node.js + npm

## 3. 文件目录（当前）

- `index.html`：官网首页（品牌介绍、优势、入口导航）
- `services.html`：主营业务详情页
- `process.html`：工艺流程与质量控制页
- `contact.html`：合作咨询页
- `src/styles.css`：全站样式
- `README.md`：项目说明、页面章节与截图
- `changelog/`：每次变更记录

## 4. 编程规范

- 保持轻量：优先原生 HTML/CSS，不引入不必要依赖
- 结构语义化：页面按 `header/main/footer` 和清晰 section 组织
- 样式复用：新增样式优先复用现有变量与通用组件（如 `.btn`、`.panel`）
- 文案风格：业务描述专业、准确，面向机械加工企业客户
- 可维护性：避免过度嵌套，命名清晰，页面跳转链接需可用

## 5. 特别说明（必须遵守）

1. **每次修改完成后**，必须在 `changelog/` 文件夹内新增或更新记录，说明本次修改点。
2. **更新 README 的页面章节**：
   - 在页面初始化或页面内容更新时，补充对应页面最新截图；
   - 需要明确说明页面间的跳转逻辑；
   - 无需每次全量更新所有页面截图，仅更新本次变更涉及页面即可。

## 6. 建议工作流

1. 修改代码与文档
2. 本地验证（`npm run build` 至少通过）
3. 更新 `changelog/` 记录
4. 更新 README 页面章节（含截图与跳转说明）
5. 提交变更
